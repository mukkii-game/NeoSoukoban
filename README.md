# ワルぶるネコのよる 🐱🕳👻

ちいさな倉庫番（そうこばん）のネコが、ひと晩かけて倉庫の「あな」をぜんぶ塞ぐ——
**倉庫番に見えるけれど、じつはルールの読み違いに気づいていく知識アンロック型パズル**です。

20〜30分で遊べる短編・全23面。PC でもスマホでも遊べます。
クリア手数で ★★★（各面）を集められます。

## 遊び方

`index.html` をブラウザで開くだけ。ビルド・サーバ不要（GitHub Pages でそのまま公開できます）。

| 操作 | キーボード | スマホ |
|------|-----------|--------|
| うごく・おす | 矢印 / WASD | スワイプ / 画面ボタン |
| まつ（その場で1ターン） | Space | 「まつ」ボタン |
| ひとつもどす (Undo) | Z | ↩ ボタン |
| やりなおす | R | ⟳ ボタン |
| レベルせんたく | Esc | 📋 ボタン |
| おと ON/OFF | M | 🔊 ボタン |

ルールはひとつだけ教えます: **📦 を押して、ぜんぶの 🕳 を塞ぐこと。**
……本当のルールは、遊びながら見つけてください。
どうしても解けないときは、しばらく粘るとネコが 💭 でつぶやきます。
それでもだめなら 💡 ボタン——**いまの状態から**解ける手順を考えて再生してくれます。
本当に詰んでいるときは、赤い ⚠️ で教えてくれます。

## 特徴

- 完全ターン制。反射神経は不要。敵の動きも1手ずつ、完全に予測可能
- **敗北はありません。** おばけは追いかけてくるだけで、なにもしてきません
- 全レベルが BFS ソルバーで「クリア可能」を機械検証済み
- 「気づき」が必要な面は「その知識なしでは解けない」ことも機械証明済み
- Undo は湧きや敵の位置まで完全に巻き戻ります。詰みを恐れず試行錯誤してください
- もし本当に詰んだら、ネコが 💭 で教えてくれます（詰みを機械的に証明できたときだけ）
- 検証用: レベル選択画面の下の「🔓 テスト: ぜんぶ ひらく」で全面を開放できます

## 開発

- `engine.js` — 純粋ロジック（ブラウザ/Node 共用）
- `levels.js` — レベルデータ
- `main.js` / `index.html` — UI・演出・入力・音
- `solutions.js` — 模範解答データ（`node tools/solve.mjs --emit` で自動生成。手で編集しない）
- `tools/solve.mjs` — レベル検証: `node tools/solve.mjs`（全レベル） / `node tools/solve.mjs n05`（個別+解表示） / `--emit`（solutions.js 再生成）
- `tools/browsertest.mjs` — ヘッドレスブラウザでの E2E（コンソールエラー0・全レベル実クリア・Undo 整合）

設計方針は `CLAUDE.md`、詳細仕様とネタバレ込みのレベル構成は `GAME_SPEC.md` を参照。
**GAME_SPEC.md は全ネタバレです。先に遊ぶのをおすすめします。**

旧作『Midnight Push Beat』は `NeoSoukoban.html` にそのまま残しています。
